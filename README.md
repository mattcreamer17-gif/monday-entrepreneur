# MondayEntrepreneur landing page

A single-file static landing page built for Maria Lopez of MondayEntrepreneur, prepared for a booth handoff at eMerge Americas 2026. The design uses MondayEntrepreneur's inferred brand tokens (navy, gold, cream, Playfair Display + Source Sans Pro) to feel native to her existing site rather than look like an Eduba pitch deck. The H1 is pulled verbatim from the fit-analysis section 7 headline, the meta description mirrors the booth opening hook, and the pitch body reflows the fit-analysis content in Maria's warm founder-brand voice. All Eduba branding is confined to the footer. The CTA routes to Matt Creamer's Calendly for a 30-minute call.
